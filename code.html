# Code

TileMill is an example of a fully javascript-based web application (both server/client side) that integrates with several external libraries including Mapnik and SQLite. It also takes advantage of several interesting APIs including `node-worker` for spawning separate processes and `backbone.js` on both the client and server. The TileMill source has been annotated using [docco](http://github.com/jashkenas/docco) for interested developers.

TileMill is released under a BSD license. Below are just some of the open source projects leveraged by TileMill:

- [Mapnik](http://www.mapnik.org) provides map rendering.
- [carto](http://github.com/mapbox/carto) provides MSS-to-Mapnik XML map style compiling.
- [tilelive.js](http://github.com/mapbox/tilelive.js) provides high-speed tile-rendering.
- [node.js](http://www.nodejs.org) provides evented server-side javascript.
- [Express](http://github.com/visionmedia/express) provides a robust HTTP web server.
- [Backbone](http://github.com/documentcloud/backbone) provides an MVC framework for client-server AJAX.
- [OpenLayers](http://www.openlayers.org) provides a tile-based map UI.
- [CodeMirror](http://www.codemirror.net) provides "in-browser code editing made bearable."

For a full list of projects see the [ndistro manifest](https://github.com/mapbox/tilemill/blob/master/.ndistro) in the TileMill repository.
